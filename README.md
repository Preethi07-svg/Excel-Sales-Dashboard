# Excel-Sales-Dashboard
Interactive Excel Sales Dashboard for business analysis.

## Project Overview

This project is an interactive Sales Dashboard created using Microsoft Excel.

The dashboard helps analyze sales performance using Pivot Tables, Pivot Charts, KPIs, and Slicers.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features

- Total Sales
- Total Profit
- Sales by Category
- Sales by Region
- Sales by Month
- Interactive Filters
